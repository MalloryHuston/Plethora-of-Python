# Wikipedia Scraper

Scrape any Wikipedia article with image (optional) without opening any browser.

## Instructions

Type ```python wiki.py --help``` to know how to use it.

```
python wiki.py "morgan freeman" --loc "c:/users/user/desktop" -i
```
