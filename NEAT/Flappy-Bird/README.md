# NEAT Flappy Bird

An AI that plays Flappy Bird! Using the NEAT Python module.

## Instructions

Run the following command ```pip install -r requirements.txt``` first!

Now, run ```python flappy_bird.py``` and watch an AI start training itself to play the game of Flappy Bird!
